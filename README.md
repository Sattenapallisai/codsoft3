# To-Do List

## Description
A Python-based command-line To-Do List application to manage daily tasks.

## Features
- Add tasks
- View tasks
- Remove tasks
- Simple menu-driven interface

## Technologies Used
- Python

## How to Run
1. Download the project.
2. Open the folder in VS Code.
3. Run:
   python todo.py
